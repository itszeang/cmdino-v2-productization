# CMDino Closed Alpha Quickstart

Follow these steps and write down anything confusing.

## 1. Install CMDino

Open the installer from the `installers` folder. If Windows shows an unsigned-app warning, continue only if you expected this CMDino alpha build.

## 2. Open CMDino

Launch CMDino from the installer, Start menu, or desktop shortcut if one was created.

## 3. Open Setup Check

Click `Setup Check`. CMDino will check which local CLI tools are installed on your machine.

It is okay if Claude, Codex, Gemini, or Ollama are missing. You can still test with a Custom Agent.

## 4. Add a Custom Agent

Choose `Start With an Agent` or `Add Agent`.

Select or create a Custom Agent with this command:

```text
cmd
```

Start the agent.

## 5. Run a Tiny Command

In the terminal, type:

```text
echo hello
```

Confirm that `hello` appears.

## 6. Try a Template

Open the template picker and load a template. Check that agents appear and the Agent Map has routes.

Some preset agents may not start if their CLI is not installed. That is okay; report whether the message is clear.

## 7. Add a Context File

Create or choose a small `.md` or `.txt` file.

Open `Add Context`, add the file, preview it, and try sending it into a running Custom Agent.

## 8. Generate Save Memory Brief

Click `Save Memory Brief`. CMDino should generate a local markdown output.

## 9. Open Output Shelf

Open `Output Shelf`, preview the generated file, and try copying text or the file path.

## 10. Save and Reopen Workspace

Save the workspace, close CMDino, reopen it, and load or continue the saved workspace.

## 11. Report One Confusing Moment

Even if nothing breaks, report one thing that was unclear, surprising, or too technical.
